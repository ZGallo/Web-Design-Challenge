Summary: Latitude vs. X

The purpose ofthis project was to analyse how wether changes as you get closer to the equator. To accomplish this analysis we first pulled data from from the OpenWeather API to assemble a dataset on over 500 cities

After assembling the dataset, we used Maplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: temperature, cloudiness, wind speed and humidity. This site provides the source data and visualizations created as part of the analyses, as well as explanations and descriptions of any trends and correlations witnessed.
